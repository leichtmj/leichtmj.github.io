Projet universitaire - 2021

Développement d'un jeu en 2D

Pour mettre en application les notions apprises pendant le 1er semestre, nous avons dû, par groupe de 3, développer un jeu en 2D en utilisant le langage C# ainsi que le framework Monogame.

Ce jeu contenait un système de combat dactylographique, où le joueur devait taper les lettres et mots affichés à l'écran, et gagnait des points en fonction de la rapidité de ses frappes.

C# - Github - Monogame

Projet universitaire - 2022

Développement d'une application de gestion des commandes

En partant d'un besoin exprimé par un client, et nécessitant une interface graphique, l'objectif était de formaliser les besoins, proposer une conception, implémenter et tester son développement.

  • Diagrammes d'analyse et de conception
  • Création d'un MCD selon les besoins
  • Développement en pattern MVC avec Data binding
  • Liaison à une base SQL Server
  • Tests unitaires
UML - C# - WPF .NET - PostgreSQL
